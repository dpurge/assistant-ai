# Anki format for research cards (stub)

This file will define:

- An Anki **notetype** suited to research facts. Likely fields:
  - `Question` — a single question.
  - `Answer` — the atomic answer.
  - `Citation` — the source citation key (e.g. `[Knuth1997]`).
  - `Topic` — the broader topic for filtering / tagging.
  - `Notes` — optional context.
- Card extraction rules from a research report (one atomic fact → one card).
- Output file format (TSV / CSV / `.apkg` via `genanki`, AGPL-3.0 — TRUE OSS).

To be filled in when `research-anki` is implemented.
